founders-components-php
=======================

Several parts of Founders' PHP implementation

*注意*
このリポジトリは、SNS "Founders"(仮称)のすべてのスクリプトを
含んでいるわけではありません。他のオープンソースソフトウェアの
ようにチェックアウト後にそのまま利用できるものとしては公開して
いませんので、ご注意ください。
このリポジトリは、各機能のうち汎用性が高いか独立して更新する
ことのできる処理を各ディレクトリに分けて開発するためのものです。
各処理は、"Founders"(仮称)のライブラリディレクトリで使用され、
そのサブディレクトリ内は"Founders"(仮称)のコア開発では一切
触れません。(依存はしますが、コア開発でそのサブディレクトリに
何かファイルを置いたりするようなことはありません)
